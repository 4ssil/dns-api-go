DNS Lookup Service

This service allows you to perform DNS-lookups via simple HTTP requests, receiving your results back as JSON-encoded objects.

The most basic usage example would look like this:

$ curl https://{{.Hostname}}/$type/$host

To discover which hosts handle email for the gmail.com domain you could make the following request:

$ curl https://{{.Hostname}}/MX/gmail.com

IPv6 lookups work exactly as you would expect, as does the lookup of TXT results too:

$ curl https://{{.Hostname}}/AAAA/dns-api.org

$ curl https://{{.Hostname}}/TXT/steve.fi

Rate-Limiting & Abuse-Protection
